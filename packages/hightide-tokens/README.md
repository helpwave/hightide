# @helpwave/hightide-tokens

Platform-agnostic **design tokens** for helpwave's _hightide_ design language.

This package is the **single source of truth** for colors, typography, spacing,
radii, elevation and motion. It has **zero runtime dependencies** and ships no
React/DOM/React-Native code, so it can be consumed by:

- `@helpwave/hightide` – the web component library (Tailwind CSS), and
- `@helpwave/hightide-native` – the React Native component library (NativeWind / `StyleSheet`).

Keeping the tokens in one place means the web and native libraries can never
visually drift apart – they are versioned and released together (see the
[architecture docs](../../docs/native/ARCHITECTURE.md)).

## Install

```bash
npm install @helpwave/hightide-tokens
```

## What's inside

| Export | Description |
| --- | --- |
| `palette` | Raw, theme-independent color scale (`gray`, `purple`, `blue`, …). |
| `lightColors` / `darkColors` / `colorSchemes` | Semantic (role-based) colors resolved per scheme. |
| `lightTheme` / `darkTheme` / `themes` | Fully-resolved theme objects (colors + typography + spacing + …). |
| `typography` | Named typographic roles (`titleMd`, `bodyLg`, …) in px. |
| `spacingScale`, `elementHeight`, … | Spacing + interactive-element sizing. |
| `radii`, `elementRadius` | Border-radius scale. |
| `elevation` | Shadows in both web (`boxShadow`) and native (`shadow*` + `elevation`) shapes. |
| `coloringRoles`, `coloringStyles`, `resolveColorRole` | The role/treatment "coloring" model used by buttons, chips, … |
| `hightidePreset` | A Tailwind / NativeWind preset derived from the tokens. |
| `createGlobalCss()`, `cssVariables` | Emits `:root` / `.dark` CSS custom properties. |

## Usage

### As a Tailwind / NativeWind preset

```js
// tailwind.config.js
const { hightidePreset } = require('@helpwave/hightide-tokens')

module.exports = {
  presets: [hightidePreset],
  content: ['./src/**/*.{ts,tsx}'],
}
```

```css
/* global.css – generated by createGlobalCss(), drives light/dark theming */
@import './hightide.generated.css';
```

### As plain values (React Native `StyleSheet`, charts, …)

```ts
import { lightTheme, resolveColorRole } from '@helpwave/hightide-tokens'

const role = resolveColorRole('primary', lightTheme.colors)
// → { color: '#694BB4', onColor: '#FFFFFF', hover: '#56389B', … }
```

## Source of truth

The values here are transcribed from the web library's CSS theme
(`src/style/theme/**`). If you change a token there, change it here too – or
better, migrate the web library to import from this package (tracked as a
follow-up in the architecture docs).
